## 프로젝트 소개 (Introduction)


**주담**은 정보의 비대칭성과 수동적인 구매 구조에 머물러 있는 전통주 시장을 혁신하기 위해 탄생했습니다. 소비자가 직접 레시피를 제안하고, 양조장은 펀딩을 통해 시장성을 검증받는 **데이터 기반의 전통주 상생 서비스**를 구축합니다.

### ** 제안 배경**
* ** 전통주 시장의 변화**: 5년 사이 시장 규모가 약 4배 급성장하며 대중화 단계 진입
* ** 규제 완화와 인프라 구축**: 소규모 양조장의 증류주 제조 허용 및 온라인 유통 활성화
* ** 소비 트렌드 확산**: 자신의 취향대로 재창조하는 **모디슈머**와 기획에 참여하는 **팬슈머**의 부상

## 기술 스택 (Technical Stack - Frontend)

### **Environment**
- **Language**: `JavaScript` / `TypeScript` (Node.js)
- **Framework**: `React Native` (Expo)
- **Editor**: `Visual Studio Code`
- **Build Engine**: `Android Studio` (Emulator: Pixel 4 API 34 등)
- **Package Manager**: `npm`

### **Libraries & Tools**
- **Navigation**: `React Navigation` / `Expo Router`
- **Styling**: `StyleSheet` / `Lucide React Native` (Icons)
- **Communication**: `Axios`
- **Visualization**: `React Native SVG` (5차원 맛 지표 차트)

## 주요 기능 (Key Features)

1. **술BTI 테스트**: 개인의 취향을 5가지 지표로 정량화하여 맞춤형 전통주 큐레이션 제공
2. **소비자 레시피 제안**: 부재료와 컨셉을 입력하면 AI가 맛 지표를 예측하고 양조장에 제안
3. **투명한 펀딩 시스템**: 제작비를 선확보하고, 앱 내 '실시간 양조 타임라인'을 통해 신뢰도 보완
4. **권한별 분기 처리**: 일반 사용자용 UI와 양조장 관리자용 대시보드(제조 현황 관리) 분리 운영
